## Getting Started
```sh
# 1. Make sure bash is installed
$ which bash
# /bin/bash

# 2. Install bats for testing; see https://bats-core.readthedocs.io/en/stable/installation.html
$ brew install bats-core # MacOS

# 3. Test the bats file; replace TEST_FILE.bats with an actual file name
$ bats TEST_FILE.bats
```
